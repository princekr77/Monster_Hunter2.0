## Monster Hunter

Monster Hunter is a 3D action game developed in Unity that focuses on player combat mechanics and intelligent enemy AI behavior within an immersive environment. The project emphasizes responsive gameplay through a custom character controller, real-time attack system, and collision-based interactions.
Designed for PC platforms, the game delivers smooth and engaging combat experiences across Windows, macOS, and Linux systems.
